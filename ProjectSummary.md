# GK Star Multi Services - Project Summary

GK Star Multi Services is a premium, modern, and responsive multi-service platform built with Next.js. It features a high-end "Black & Gold" luxury aesthetic, designed to showcase bespoke tailoring, creative branding, and professional lifestyle services.

## 🚀 Technologies Used
- **Framework**: Next.js 16.1.7 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4.x
- **Icons**: Lucide React & Font Awesome
- **Development**: Turbopack

## ✨ Key Features
- **Cinematic Hero Section**: A high-impact landing with smooth animations and premium typography.
- **Dynamic Service Showcasing**: Featured categories for Tailoring, Aari Work, and Designing.
- **Admin Dashboard**: Comprehensive management interface for items and services.
- **Mobile Responsive**: Fully optimized for all device sizes.
- **WhatsApp Integration**: Fast enquiry button for direct client communication.
- **Luxury Aesthetic**: Curated color palette (Gold `#D4AF37`) and glassmorphism effects.

## 📂 Project Structure
- `/src/app`: App Router with pages for Home, Services, Gallery, About, and Contact.
- `/src/components`: Reusable UI components including a custom Admin Dashboard.
- `/public`: Static assets and media.

## 🛠️ How to Run Locally
To start the development server:
```bash
npm install
npx next dev -p 3001
```
*Note: Port 3001 is recommended to avoid workspace root conflicts.*
